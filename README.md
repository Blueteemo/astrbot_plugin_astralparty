# 《吉星派对》随机挑战（astrbot_plugin_astralparty）

随机抽取游戏《吉星派对》中的地图与角色，支持单抽、组队、组合三种模式。

## 功能

| 命令 | 说明 | 示例输出 |
|------|------|---------|
| `星趴角色` | 随机抽取一个角色 | 你抽取到的角色是：太刀 |
| `星趴队伍` | 随机抽取一支四人队伍 | 队伍：太刀 / 风水 / 猫猫 / 茉莉 |
| `星趴随机` | 随机抽取地图 + 四人队伍 | 你抽取到的地图是：星趴·梦想号 |

## 安装

1. 将插件目录 `astrbot_plugin_astralparty` 放入 AstrBot 插件目录
2. 重启 AstrBot

## 环境依赖

- AstrBot
- Python 3.10+
